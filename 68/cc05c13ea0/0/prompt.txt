**Prerequisites**
- Rust toolchain (for building)
- `curl` (used internally for Slack API calls)

**Installation**
```bash
cargo install --path .
```

**Setup (Slack App)**
1. Create a Slack app at https://api.slack.com/apps
2. Add OAuth redirect URL: `https://127.0.0.1:9876`
3. Add User Token Scopes: `channels:history`, `groups:history`, `users:read`
4. Note the Client ID and Client Secret

**Configuration**
- Environment variables: `SLK_CLIENT_ID`, `SLK_CLIENT_SECRET`
- Or config file: `~/.config/slk/config.json`
  ```json
  { "client_id": "...", "client_secret": "..." }
  ```
- Then run `slk login` to authenticate (token saved to `~/.config/slk/credentials`)
- Alternatively, set `SLACK_TOKEN` env var directly
